# Titanic-Survival
This was my final project for my predictive modeling class
This was created in Python
The dataset used is the Titanic dataset from Kaggle 
Modeling in this dataset includes:
  1.KNN
  2.Decision Tree
  3.Random Forest
  4.SVM-Linear
  5.SVM-RBF
  6.Stochastic Gradient Descent
  7.Adaboost
  8.Bagging
  9.Gradient Boosting
  10.Extra Trees
